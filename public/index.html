¿Qué permisos solicitamos?

Al iniciar sesión, solicitamos los siguientes permisos:

  • user:read - Para acceder a información básica de tu perfil
  • channel:read - Para acceder a información de tu canal
  • events:subscribe - Para recibir notificaciones de eventos
  • channel:write - Para realizar cambios en tu canal
  • chat:write - Para enviar mensajes en el chat

Puedes revocar estos permisos en cualquier momento desde tu cuenta de Kick.

Esta aplicación utiliza el flujo OAuth 2.1 para autenticación segura con Kick.com
